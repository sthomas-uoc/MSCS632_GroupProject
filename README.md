# Code for MSCS 632 Group Project - Deliverable 1
Implement Expense Tracker in Python and C++.

## Build and run instructions

### C++
The C++ implementation is in the file `cppexpense/cpp_expense.cc`
A C++ 11 compatible compiler with STL is required.

To build the project on a Linux system, run `g++ -o cppexpense/cpp_expense cppexpense/cpp_expense.cc`

To run the project, run `./cppexpense/cpp_expense`

### Python
The Python implementation is in the file expensetracker.py
As this is a Python program, Python version 3 or higher is required.

To run the project run `python expensetracker.py`
